    Timeline

    Play the timeline: menubar above timeline list

    Map

    Change map layers: top left layers button on map

    Edit legend labels: double click label text

    Save and Load

    Warning: opening a save, loading new basemap wipes clean current map

    The left most three new map options load in a new base map and wipes the current one clean

    New Map from Files require user to upload their own geojson basemaptom geojson basemap

    Colouring

    Use single fill for filling single province, use lasso fill to fill or delete multiple provinces

    Currently chosen colour will have a solid black border and be protruding

    Colour of currently hovered over region will have a dashed black border and be protruding

    Colours in the toolbar will have a double border if they are on the map already

    Choose a colour: either from colour bar above map or from legend

    Colour region: choose a colour and a date, then simply click the region
